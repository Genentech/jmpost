# `LinkComponent` -\> `list`

Returns a named list where each element of the list corresponds to a
Stan modelling block e.g. `data`, `model`, etc.

## Usage

``` r
# S3 method for class 'LinkComponent'
as.list(x, ...)
```

## Arguments

- x:

  (`LinkComponent`) a link component.

- ...:

  Not Used.
