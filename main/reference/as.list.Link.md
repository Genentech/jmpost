# `Link` -\> `list`

Returns a named list where each element of the list corresponds to a
Stan modelling block e.g. `data`, `model`, etc.

## Usage

``` r
# S3 method for class 'Link'
as.list(x, ...)
```

## Arguments

- x:

  ([`Link`](https://genentech.github.io/jmpost/reference/Link-class.md))\
  a link object.

- ...:

  Not Used.
