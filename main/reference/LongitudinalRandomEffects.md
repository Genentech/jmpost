# Extract Random Effects Samples from a Longitudinal Model

Helper function to extract subject-level random effects samples from the
longitudinal sub-model of a joint model samples object.

## Usage

``` r
LongitudinalRandomEffects(object)
```

## Arguments

- object:

  (`JointModelSamples`) samples as drawn from a Joint Model.
