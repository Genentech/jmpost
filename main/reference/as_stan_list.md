# `as_stan_list`

Extracts a list of data elements from an object to be used as input to a
Stan Model

## Usage

``` r
as_stan_list(object, ...)

# Default S3 method
as_stan_list(object, ...)
```

## Arguments

- object:

  to be converted.

- ...:

  additional options.
