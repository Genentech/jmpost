# `RandomEffectQuantities` -\> `data.frame`

Returns a `data.frame` of the subject-level random effect parameter
samples.

## Usage

``` r
# S3 method for class 'RandomEffectQuantities'
as.data.frame(x, ...)
```

## Arguments

- x:

  (`RandomEffectQuantities`) generated quantities.

- ...:

  not used.
