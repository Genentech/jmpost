# Resolve a Promise

Resolve a Promise

## Usage

``` r
resolvePromise(object, ...)

# Default S3 method
resolvePromise(object, ...)
```

## Arguments

- object:

  (`ANY`)\
  an object to resolve.

- ...:

  (`ANY`)\
  additional arguments.

  If `object` is not a promise will just return itself else will resolve
  the promise and return the promised object.
