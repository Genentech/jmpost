# Abstract Simulation Class for Longitudinal Data

This class exists to be extended by other classes that simulate
longitudinal data. It is not intended to be used directly.

## Usage

``` r
SimLongitudinal(times = seq(0, 100, 50))
```

## Arguments

- times:

  (`numeric`) the times to generate observations at.
