# Claret-Bruno Functionals

Claret-Bruno Functionals

## Usage

``` r
clbr_sld(t, b, g, c, p)

clbr_ttg(t, b, g, c, p)

clbr_dsld(t, b, g, c, p)
```

## Arguments

- t:

  (`numeric`) time grid.

- b:

  (`number`) baseline sld.

- g:

  (`number`) growth rate.

- c:

  (`number`) resistance rate.

- p:

  (`number`) growth inhibition.

## Value

The function results.
