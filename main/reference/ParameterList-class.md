# `ParameterList`

This class extends the general
[`list`](https://rdrr.io/r/base/list.html) type for containing
[`Parameter`](https://genentech.github.io/jmpost/reference/Parameter-class.md)
specifications.

## Usage

``` r
ParameterList(...)
```

## Arguments

- ...:

  (`Parameter`)\
  which parameter specifications to include.

## Slots

- `parameters`:

  (`list`)\
  a list of
  [`Parameter`](https://genentech.github.io/jmpost/reference/Parameter-class.md)
  objects
