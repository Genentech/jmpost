# Log-Logistic Prior Distribution

Log-Logistic Prior Distribution

## Usage

``` r
prior_loglogistic(alpha, beta)
```

## Arguments

- alpha:

  (`number`) Scale parameter.

- beta:

  (`number`) Shape parameter.
