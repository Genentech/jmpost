# `DataLongitudinal` -\> `data.frame`

Converts a
[`DataLongitudinal`](https://genentech.github.io/jmpost/reference/DataLongitudinal-class.md)
object into a `data.frame`. The subject variable is cast to factor.

## Usage

``` r
# S3 method for class 'DataLongitudinal'
as.data.frame(x, ...)
```

## Arguments

- x:

  (`DataLongitudinal`) Longitudinal Data.

- ...:

  Not Used.
