# `Parameter` -\> `list`

Converts a Parameter object to a list of parameter data values for a
Stan model.

## Usage

``` r
# S3 method for class 'Parameter'
as_stan_list(object, ...)
```

## Arguments

- object:

  (`Parameter`) a prior Distribution

- ...:

  Not Used.
