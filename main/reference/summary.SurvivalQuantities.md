# summary

This method returns a `data.frame` of the longitudinal quantities.

## Usage

``` r
# S3 method for class 'SurvivalQuantities'
summary(object, conf.level = 0.95, ...)
```

## Arguments

- object:

  (`SurvivalQuantities`) survival quantities.

- conf.level:

  (`numeric`) confidence level of the interval.

- ...:

  not used.
