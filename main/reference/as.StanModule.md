# `as.StanModule`

Converts an object into a
[`StanModule`](https://genentech.github.io/jmpost/reference/StanModule-class.md).

## Usage

``` r
as.StanModule(object, ...)
```

## Arguments

- object:

  what to convert.

- ...:

  additional options.
