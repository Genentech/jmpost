# `DataLongitudinal` -\> Printable `Character`

Converts
[`DataLongitudinal`](https://genentech.github.io/jmpost/reference/DataLongitudinal-class.md)
object into a printable string.

## Usage

``` r
# S3 method for class 'DataLongitudinal'
as_print_string(object, indent = 1, ...)
```

## Arguments

- object:

  ([`DataLongitudinal`](https://genentech.github.io/jmpost/reference/DataLongitudinal-class.md))\
  Longitudinal Data.

- indent:

  (`numeric`)\
  how much white space to prefix the print string with.

- ...:

  Not Used.
