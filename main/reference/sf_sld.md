# Stein-Fojo Functionals

Stein-Fojo Functionals

## Usage

``` r
sf_sld(time, b, s, g)

sf_ttg(time, b, s, g)

sf_dsld(time, b, s, g)
```

## Arguments

- time:

  (`numeric`) time grid.

- b:

  (`number`) baseline.

- s:

  (`number`) shrinkage.

- g:

  (`number`) growth.

## Value

The function results.
