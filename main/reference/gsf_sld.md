# Generalized Stein-Fojo Functionals

Generalized Stein-Fojo Functionals

## Usage

``` r
gsf_sld(time, b, s, g, phi)

gsf_ttg(time, b, s, g, phi)

gsf_dsld(time, b, s, g, phi)
```

## Arguments

- time:

  (`numeric`) time grid.

- b:

  (`number`) baseline.

- s:

  (`number`) shrinkage.

- g:

  (`number`) growth.

- phi:

  (`number`) shrinkage proportion.

## Value

The function results.
