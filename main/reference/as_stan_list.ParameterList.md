# `ParameterList` -\> `list`

Converts a ParameterList object to a list of parameter data values for a
Stan model.

## Usage

``` r
# S3 method for class 'ParameterList'
as_stan_list(object, ...)
```

## Arguments

- object:

  (`ParameterList`)\
  A List of
  [`Parameter`](https://genentech.github.io/jmpost/reference/Parameter-class.md)
  Objects.

- ...:

  Not Used.
