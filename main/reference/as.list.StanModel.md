# `StanModel` -\> `list`

Returns a named list where each element of the list corresponds to a
Stan modelling block e.g. `data`, `model`, etc.

## Usage

``` r
# S3 method for class 'StanModel'
as.list(x, ...)
```

## Arguments

- x:

  ([`StanModel`](https://genentech.github.io/jmpost/reference/StanModel-class.md))\
  A Stan Model

- ...:

  Not Used.
