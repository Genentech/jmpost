# `DataSubject` -\> Printable `Character`

Converts
[`DataSubject`](https://genentech.github.io/jmpost/reference/DataSubject-class.md)
object into a printable string.

## Usage

``` r
# S3 method for class 'DataSubject'
as_print_string(object, indent = 1, ...)
```

## Arguments

- object:

  ([`DataSubject`](https://genentech.github.io/jmpost/reference/DataSubject-class.md))\
  subject-level data.

- indent:

  (`numeric`)\
  how much white space to prefix the print string with.

- ...:

  Not Used.
