# `JointModelSamples` -\> Printable `Character`

Converts
[`JointModelSamples`](https://genentech.github.io/jmpost/reference/JointModelSamples-class.md)
object into a printable string.

## Usage

``` r
# S3 method for class 'JointModelSamples'
as_print_string(object, indent = 1, ...)
```

## Arguments

- object:

  (`JointModelSamples`) samples as drawn from a
  [`JointModel`](https://genentech.github.io/jmpost/reference/JointModel-class.md).

- indent:

  (`numeric`) how much white space to prefix the print string with.
