# Coalesce Time

Coalesce Time

## Usage

``` r
coalesceGridTime(object, times, ...)

# S3 method for class 'GridFixed'
coalesceGridTime(object, times, ...)

# S3 method for class 'GridGrouped'
coalesceGridTime(object, times, ...)

# S3 method for class 'GridPopulation'
coalesceGridTime(object, times, ...)

# S3 method for class 'GridPrediction'
coalesceGridTime(object, times, ...)
```

## Arguments

- object:

  (`Grid`) object to coalesce time for.

- times:

  (`numeric`) the times to coalesce to.

- ...:

  Not used

  Method used to replace NULL times on grid objects (if appropriate)
