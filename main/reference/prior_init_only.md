# Initial Values Specification

This function is used to specify only the initial values for a
parameter. This is primarily used for hierarchical parameters whose
distributions are fixed within the model and cannot be altered by the
user.

## Usage

``` r
prior_init_only(dist)
```

## Arguments

- dist:

  (`Prior`) a prior Distribution
