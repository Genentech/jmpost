# `DataSurvival` -\> Printable `Character`

Converts
[`DataSurvival`](https://genentech.github.io/jmpost/reference/DataSurvival-class.md)
object into a printable string.

## Usage

``` r
# S3 method for class 'DataSurvival'
as_print_string(object, indent = 1, ...)
```

## Arguments

- object:

  (`DataSurvival`) Survival Data.

- indent:

  (`numeric`) how much white space to prefix the print string with.

- ...:

  Not Used.
