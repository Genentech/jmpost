# `LinkComponent` Function Arguments

This exists to contain all the common arguments for
[`LinkComponent`](https://genentech.github.io/jmpost/reference/LinkComponent-class.md)
methods.

## Arguments

- stan:

  (`StanModule`)\
  Stan code.

- x:

  ([`LinkComponent`](https://genentech.github.io/jmpost/reference/LinkComponent-class.md))\
  a link component.

- object:

  ([`LinkComponent`](https://genentech.github.io/jmpost/reference/LinkComponent-class.md))\
  a link component.

- ...:

  Not Used.
