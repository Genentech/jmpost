# summary

This method returns a `data.frame` of the longitudinal quantities.

## Usage

``` r
# S3 method for class 'LongitudinalQuantities'
summary(object, conf.level = 0.95, ...)
```

## Arguments

- object:

  ([`LongitudinalQuantities`](https://genentech.github.io/jmpost/reference/LongitudinalQuantities-class.md))\
  longitudinal quantities.

- conf.level:

  (`numeric`)\
  confidence level of the interval.

- ...:

  not used.
